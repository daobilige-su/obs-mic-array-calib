This is supplementary code for our paper 
---
Daobilige Su, He Kong, Salah Sukkarieh, Shoudong Huang, “Necessary and Sufficient Conditions for Observability of SLAM-based Microphone Array Calibration and Sound Source Localization”.
---

To run code, simply execute “main.m” file in Matlab. Tested in Matlab2018B, but should work with all Matlab version after 2015A.

After successful execution, The information shown in “results.txt” file should show on Matlab console.

The code partially based on SLAM open course provided by Prof. Dr. Cyrill Stachniss from University of Bonn, Germany.
